Matn Retriever

This dashboard retrieves the most similar matns from the database to the input text. Similarity is achieved by using cosine similarity of an embedding, using our QulBERT Embedding Model. It returns the matn, Taraf Id, book name, book author, Similarity score, and Hadith ID. If the dashboard says "preparing space..." it is loading because it has not been used and must be reloaded. Give it a minute for it to load.
